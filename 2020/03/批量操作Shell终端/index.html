批量操作Shell终端

发表于 2020-03-08 更新于 2025-08-23 分类于 Tools ， SSH 阅读次数： Waline：

在生产环境，往往有几十上百台服务器要同时操作，如何同时控制这么多服务器呢？

Windows

在Windows环境下，可以使用XShell。

点击工具(T)->发送键到所有会话(K)

MacOS

可以使用 iTerm2 终端工具，快捷键是Command + Shift + I。然后 iTerm2 窗口会出现背景斜条纹。

# linux # shell # terminal # xshell # iterm2
