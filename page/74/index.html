/var/lib/docker/overlay2占用空间太大

发表于 2020-02-27 更新于 2025-08-23 分类于 Docker 阅读次数： Waline：

执行df -h命令发现/var/lib/docker目录占用空间太大。

执行docker ps -a发现大量已经停止或退出的容器，需要清理空间。

这样就删除了不用的容器和镜像了。
