安装git

打开 Git Bash

这个邮箱可以随便输入的，并不需要是 Github.com 的登录帐户。

拷贝 id_rsa.pub 内容到 Github

Settings -> SSH and GPG keys -> New SSH key

把前面生成的 /c/Users/simon/.ssh/id_rsa.pub 文件内容粘贴上去就可以了。
