VMware虚拟机Ubuntu设置自适应屏幕大小

发表于 2019-10-20 更新于 2025-08-23 分类于 OS ， Ubuntu 阅读次数： Waline：

在VMware上安装了Ubuntu虚拟机后，发现屏幕显示得比较别扭，需要通过安装VMware Tools，以达到屏幕自适应宽度。

点击菜单，虚拟机 -> 安装VMware Tools

此时CD/DVD设置将自动连接到linux.iso镜像。

把安装软件VMwareTools-10.0.10-4301679.tar.gz拷贝到磁盘中比如/tmp目录。

解压

运行vmware-install.pl文件

然后一路回车，默认选项，最后重新登录，就可以达到Ubuntu屏幕自适应大小了。
