NIO的通道

发表于 2019-10-14 更新于 2025-08-23 分类于 Java ， NIO 阅读次数： Waline：

通道(Channel)，用于源节点和目标节点的连接。在Java NIO中负责数据的传输。Channel本身不存储数据，所以传输时还需要缓冲区配合。
