安装Visual GC插件

点击 工具 -> 插件

选择设置标签，更新中心配置，也就是设置从哪里下载插件。

可以在 https://visualvm.github.io/pluginscenters.html 页面找到不同版本的插件源地址。

设置好以后，在可用插件标签下面，我们选择安装Visual GC插件。

安装好以后，重启Java VisualVM
